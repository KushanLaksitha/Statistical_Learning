Cross-Validation & Bootstrap

Model evaluation, hyperparameter tuning, uncertainty estimation. Q5 exam fully covers this.

🔄 Cross-Validation (CV)

ඇයි CV?
Test set error estimate කිරීමට. Model selection (polynomial degree, λ, k) සඳහා. Single train/test split variable — CV more reliable.

k-Fold Cross-Validation

Data k equal folds divide. k-1 folds train, 1 fold test. k iterations rotate. Final CV error = mean(fold errors).

CV Error = (1/K) Σ MSEₖ
Method Description Pros Cons
LOOCV k = n (leave one out) Unbiased Computationally slow
5-Fold CV k = 5 Balance Some variance
10-Fold CV k = 10 Standard choice Slightly slower than 5-fold
R – 10-Fold CV (from rmd.txt)
R
set.seed(123)
df <- read.csv("./data.set.1.csv", header = TRUE)
n <- dim(df)[1]
K <- 10
folds <- sample(rep(1:K, length.out = n))

cv_error <- function(degree) {
  mse <- numeric(K)
  for (k in 1:K) {
    train <- df[folds != k, ]
    test  <- df[folds == k, ]
    fit   <- lm(y ~ poly(x, degree, raw = TRUE), data = train)
    pred  <- predict(fit, newdata = test)
    mse[k] <- mean((test$y - pred)^2)
  }
  mean(mse)
}

degrees    <- 1:25
cv_results <- sapply(degrees, cv_error)
best_degree <- degrees[which.min(cv_results)]
cat("Best polynomial degree:", best_degree)

🥾 Bootstrap

Bootstrap ගැළවීම
Population distribution නොදැනත්, statistic හි variability estimate කිරීමට. Replace ගෙන resample → empirical sampling distribution.

Bootstrap Ideal ෙදී (Q text source)

  • No closed-form SE: Median, quantiles, trimmed mean — theoretical SE formula නැත
  • Assumption violations: Normality, homoscedasticity violate — bootstrap only needs i.i.d.
  • Small samples: Asymptotic approximations unreliable
  • Complex estimators: ML performance metrics, AUC
  • Non-parametric CIs: Skewness/asymmetry reflect

Bootstrap Algorithm

  1. Original sample x (size n) select
  2. B = 2000 times: Sample n observations with replacement
  3. Each bootstrap sample ෙදී statistic calculate
  4. B statistics distribution = bootstrap distribution
  5. SE, CI ගෙන estimate
Bootstrap SE = sd(bootstrap_statistics)
95% CI = quantile(bootstrap_stats, c(0.025, 0.975))

📝 Q5 Complete Solution (Exam)

Q5: Bootstrap + Linear Regression
Generate Y = 3 + 2X + ε, ε ~ N(0,4). Bootstrap 1000 replications. SE and CI for β₁.
R – Q5 Complete Answer (Rmd)
R
# Q5(a): Generate dataset (100 observations)
set.seed(123)
n   <- 100
X   <- runif(n, 0, 10)          # Xi ~ Unif(0, 10)
eps <- rnorm(n, 0, 2)           # εi ~ N(0, 4) [sd=2]
Y   <- 3 + 2 * X + eps           # Yi = 3 + 2Xi + εi
df5 <- data.frame(X = X, Y = Y)

# Q5(b): Scatter plot
plot(df5, pch = 16, col = "blue",
     main = "Y vs X Scatter Plot",
     xlab = "X", ylab = "Y")
abline(lm(Y ~ X, data = df5), col = "red", lwd = 2)

# Q5(c): Fit linear regression
fit5 <- lm(Y ~ X, data = df5)
summary(fit5)
beta0_hat <- coef(fit5)[1]  # Intercept β̂₀
beta1_hat <- coef(fit5)[2]  # Slope β̂₁
cat("β̂₀ =", beta0_hat, "\nβ̂₁ =", beta1_hat)

# Q5(d): Nonparametric Bootstrap (1000 replications)
B <- 1000
boot_results <- data.frame(
  b       = 1:B,
  beta0_b = numeric(B),
  beta1_b = numeric(B)
)

for (b in 1:B) {
  idx         <- sample(1:n, size = n, replace = TRUE)
  boot_sample <- df5[idx, ]
  boot_fit    <- lm(Y ~ X, data = boot_sample)
  boot_results$beta0_b[b] <- coef(boot_fit)[1]
  boot_results$beta1_b[b] <- coef(boot_fit)[2]
}

# Bootstrap SE for β₁
boot_se_beta1 <- sd(boot_results$beta1_b)
cat("Bootstrap SE for β₁:", boot_se_beta1)

# Q5(e): Bootstrap distribution histogram
hist(boot_results$beta1_b, breaks = 30, col = "skyblue",
     main = "Bootstrap Distribution of β̂₁",
     xlab = "Slope (β₁)")
abline(v = beta1_hat, col = "blue", lwd = 2)

# Q5(f): 95% Percentile-based CI
boot_ci_beta1 <- quantile(boot_results$beta1_b, c(0.025, 0.975))
cat("95% Bootstrap CI for β₁:", boot_ci_beta1)

Bootstrap Examples

R – Bootstrap Mean & Median (from rmd.txt)
R
set.seed(123)
x <- rexp(50, rate = 0.5)  # Exponential distribution
B <- 2000; n <- length(x)

boot_mean <- boot_median <- numeric(B)

for (b in 1:B) {
  idx  <- sample(1:n, size = n, replace = TRUE)
  xb   <- x[idx]
  boot_mean[b]   <- mean(xb)
  boot_median[b] <- median(xb)
}

# SEs and CIs
data.frame(
  Statistic = c("Mean", "Median"),
  Estimate  = c(mean(x), median(x)),
  Boot_SE   = c(sd(boot_mean), sd(boot_median)),
  CI_Lower  = c(quantile(boot_mean,  0.025),
                quantile(boot_median, 0.025)),
  CI_Upper  = c(quantile(boot_mean,  0.975),
                quantile(boot_median, 0.975))
)
