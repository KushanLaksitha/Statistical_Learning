Linear Regression

Continuous outcome predict කිරීමට ඇති ශ්‍රේෂ්ඨ method. OLS, Assumptions, Multiple Regression, KNN.

📈 Simple Linear Regression

නිර්වචනය
Simple Linear Regression: Y සහ X අතර linear සම්බන්ධය model කිරීම. Predictor (X) එකක් පමණි.
Ŷ = β̂₀ + β̂₁X
β̂₀ = intercept · β̂₁ = slope

Interpretation (Q1(c) exam answer):

  • β̂₀ (Intercept): X = 0 ලෙස X = 0 ලෙස X = 0 ලෙස Y හි estimated value. (e.g., Temperature = 0 ෙදී Relative Humidity)
  • β̂₁ (Slope): X හි 1 unit increase ෙදී Y හි average change. e.g., "Temperature 1°C increase ෙදී RH -2.5% change"

🔢 OLS Estimates (Ordinary Least Squares)

OLS method: Residual Sum of Squares (RSS) minimize කිරීම.

RSS = Σ(yᵢ − ŷᵢ)² → Minimize!

β̂₁ = Σ(xᵢ − x̄)(yᵢ − ȳ) / Σ(xᵢ − x̄)²
β̂₀ = ȳ − β̂₁x̄
📊 Model Evaluation:
  • R²: Variance explained (0 to 1). R² = 0.85 → 85% explained.
  • RSE (Residual Standard Error): Average residual size.
  • p-value: Coefficient significant ද? (p < 0.05 → significant)

✅ OLS Assumptions (LINE)

L – Linearity

X සහ Y අතර linear සම්බන්ධය.
Check: Residuals vs Fitted plot

I – Independence

Observations independent.
Check: Durbin-Watson test

N – Normality

Errors normally distributed.
Check: Q-Q plot, Shapiro-Wilk

E – Equal Variance

Homoscedasticity.
Check: Residuals vs Fitted – no funnel

📊 Multiple Linear Regression

Y = β₀ + β₁X₁ + β₂X₂ + ... + βₚXₚ + ε

Predictors කිහිපයක් ඇති විට. Each βⱼ = "other predictors constant holding, Xⱼ හි 1 unit change ෙදී Y change".

Stepwise Variable Selection:

  • Forward: Predictors gradually add කිරීම. Empty → Full.
  • Backward: Full model සිට insignificant predictors remove.
  • Criterion: AIC, BIC, Adjusted R²

🔵 KNN Regression

Linear Regression

  • Parametric model
  • Interpretable
  • Assumes linearity
  • Good when linear
  • Extrapolation possible

KNN Regression

  • Non-parametric
  • Less interpretable
  • No functional form
  • Good for non-linear
  • Curse of dimensionality
⚖️ K selection: Small k → Low bias, High variance (overfit). Large k → High bias, Low variance (underfit). Cross-validation ගෙන optimal k select.

💻 R Code – Q1 Air Quality

R – Q1(c): Simple Linear Regression
R
# Part (a): Load dataset
df <- read.csv("AirQuality.csv", header = TRUE)
str(df); summary(df)

# Part (b): Visualization
library(corrplot)
nums <- df[, sapply(df, is.numeric)]
corrplot(cor(nums, use = "complete.obs"), method = "color")
pairs(nums[, c("RH","T","AH","CO")], pch = 16, col = "blue")

# Part (c): Simple Linear Regression RH ~ T
fit_simple <- lm(RH ~ T, data = df)
summary(fit_simple)
# Regression equation: RH = β̂₀ + β̂₁ × T

# Part (d): Multiple Linear Regression
fit_mlr <- lm(RH ~ ., data = df)
summary(fit_mlr)
# p-value < 0.05 olan predictors = significant

# Part (e): Stepwise Selection
fit_step <- step(fit_mlr, direction = "backward")
summary(fit_step)
R – Q1(f): KNN Regression
R
library(caret)

# Data prep
df_clean <- na.omit(df)
X <- df_clean[, !names(df_clean) %in% "RH"]
Y <- df_clean$RH

# Train/Test split 70/30
set.seed(42)
train_idx <- createDataPartition(Y, p = 0.7, list = FALSE)
X_train <- X[train_idx, ]; Y_train <- Y[train_idx]
X_test  <- X[-train_idx, ]; Y_test  <- Y[-train_idx]

# Standardize predictors
pp <- preProcess(X_train, method = c("center","scale"))
X_train_sc <- predict(pp, X_train)
X_test_sc  <- predict(pp, X_test)

# KNN with CV to find optimal k
ctrl <- trainControl(method = "cv", number = 10)
knn_model <- train(
  x = X_train_sc, y = Y_train,
  method = "knn",
  trControl = ctrl,
  tuneLength = 20
)
print(knn_model$bestTune)

# Test set RMSE
preds <- predict(knn_model, X_test_sc)
rmse  <- sqrt(mean((Y_test - preds)^2))
cat("KNN Test RMSE:", rmse)
