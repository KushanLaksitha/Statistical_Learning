Statistical Learning Introduction

Data සමඟ ගනුදෙනු කිරීමේ විද්‍යාව. Prediction, Inference, සහ Model Complexity.

🔭 Statistical Learning යනු කුමක්ද?

නිර්වචනය
Statistical Learning යනු data අර්ථ ගැනීම සහ predictions කිරීම සඳහා භාවිත කරන mathematical tools සහ algorithms සමූහයකි. සාම්ප්‍රදායික statistics සහ machine learning දෙකම ඒකාබද්ධ කරයි.

Statistical Learning හි ප්‍රධාන ඉලක්ක දෙකක් ඇත:

  • Prediction (අනාවැකිය): නව data points සඳහා Y (output) predict කිරීම. උදාහරණ: ගෙවල් මිළ predict කිරීම.
  • Inference (අනුමානය): Predictors සහ Response අතර සම්බන්ධය තේරුම් ගැනීම. X ශ්‍රිතය Y ට කෙසේ බලපෑමද?
ලක්ෂණය සාම්ප්‍රදායික Statistics Statistical Learning
ප්‍රධාන ඉලක්කය Inference Prediction
Model Complexity Simple, interpretable Flexible, may be complex
Data ප්‍රමාණය Low-dimensional Often high-dimensional
Assumptions Strong (normality, linearity) Flexible or minimal
Output Coefficients, p-values Predictive accuracy, CV

🗂️ ඉගෙනීමේ ආකාර

✅ Supervised Learning

Response variable (Y) ඇත

  • Regression: Continuous Y — mpg predict කිරීම
  • Classification: Categorical Y — email spam/not spam
  • Algorithm: Linear Regression, Logistic Regression, KNN, SVM

🔍 Unsupervised Learning

Response variable නැත

  • Clustering: Data groups discover කිරීම
  • PCA: Dimension reduction
  • Algorithm: K-means, Hierarchical Clustering
📝 Exam Tip: Supervised learning හි goal කුමක්ද? → Y predict කිරීම labeled data භාවිතයෙන්. Unsupervised → structure discover කිරීම.

⚖️ Bias-Variance Tradeoff

Model error = Bias² + Variance + Irreducible Error

E[(Y − f̂(x₀))²] = [Bias(f̂(x₀))]² + Var(f̂(x₀)) + σ²

📉 High Bias (Underfitting)

  • Model too simple
  • Training error ↑, Test error ↑
  • Pattern miss කරයි
  • Example: 12 degree data fit කිරීමට linear model

📈 High Variance (Overfitting)

  • Model too complex
  • Training error ↓, Test error ↑
  • Noise fit කරයි
  • Example: 3 data points fit කිරීමට 10th degree polynomial
⚠️ Key Insight: Model complexity වැඩි කළ විට Bias ↓ නමුත් Variance ↑. Optimal model = Bias² + Variance minimize කරන point.
✅ Goal: Test MSE minimize කරන model select කිරීම. Cross-validation මෙය estimate කිරීමට help කරයි.

💻 R Code Examples

R – Supervised Learning (Train/Test Split)
R 
# ===== SUPERVISED LEARNING EXAMPLE =====
library(caret)

# Data load කිරීම
data(Auto)

# Train/Test split (70/30)
set.seed(42)
train_idx <- createDataPartition(Auto$mpg, p = 0.7, list = FALSE)
train_data <- Auto[train_idx, ]
test_data  <- Auto[-train_idx, ]

# Linear Regression model fit
model <- lm(mpg ~ horsepower, data = train_data)

# Predictions
pred <- predict(model, test_data)

# RMSE calculate
rmse <- sqrt(mean((test_data$mpg - pred)^2))
cat("RMSE:", rmse)
R – Unsupervised Learning (K-means)
R 
# ===== UNSUPERVISED LEARNING (K-means) =====
data(Auto)

# Cluster mpg සහ horsepower සඳහා
set.seed(42)
km <- kmeans(Auto[, c("mpg", "horsepower")], centers = 3)

plot(Auto$horsepower, Auto$mpg,
     col = km$cluster, pch = 16,
     main = "K-means Clustering")
points(km$centers, col = 1:3, pch = 8, cex = 2)
