Regularization: Ridge & Lasso

Overfitting prevent කිරීමට Penalty terms. Ridge L2, Lasso L1, Elastic Net.

❓ Regularization ඇයි?

OLS problems:

  • Multicollinearity: Correlated predictors → Unstable coefficients
  • Overfitting: Too many predictors → Poor generalization
  • p ≈ n or p > n: More predictors than observations
Regularization
Loss function ෙදී penalty term add කිරීම. β coefficients toward zero shrink කරයි. Model complexity control.

🔵 Ridge Regression (L2 Penalty)

Ridge: min[Σ(yᵢ − ŷᵢ)² + λΣβⱼ²]
= RSS + λ × (β squared sum)
  • β coefficients shrink toward zero නමුත් exactly zero නොවේ
  • Correlated predictors together shrink → stable
  • Variable selection නොකරයි
  • λ = 0: No penalty = OLS. λ → ∞: β → 0

🟠 Lasso Regression (L1 Penalty)

Lasso: min[Σ(yᵢ − ŷᵢ)² + λΣ|βⱼ|]
= RSS + λ × (β absolute sum)
  • Some β coefficients exactly zero → Variable selection!
  • Sparse model → Interpretable
  • Correlated predictors → Arbitrarily one select
  • High-dimensional data (p >> n) ideal
📝 Q2(a) Exam Answer: Ridge vs Lasso

Ridge: L2 penalty (β²). Coefficients shrink toward zero නමුත් never exactly zero. Multicollinearity handle. Variable selection නොකරයි.

Lasso: L1 penalty (|β|). Some coefficients exactly zero → Automatic variable selection. Sparse models produce. Correlated predictors සමඟ unstable.

🔴 Elastic Net

min[Σ(yᵢ − ŷᵢ)² + λ(α Σ|βⱼ| + (1−α) Σβⱼ²)]
α=1: Lasso · α=0: Ridge · 0<α<1: Both

Ridge + Lasso combine. Correlated predictors group together select. Tuning: λ (strength) + α (mix).

Method Penalty Variable Selection Correlated Predictors
Ridge β² No (shrinks, not zero) Good — groups together
Lasso |β| Yes (some exactly zero) Unstable — picks one
Elastic Net Both Yes + grouping Best — groups + selects

λ Tuning (Cross-Validation)

  • Large λ: More penalty → Coefficients toward zero (underfitting risk)
  • Small λ: Less penalty → Many features retained (overfitting risk)
  • cv.glmnet() ගෙන optimal λ select

💻 R Code – Q1(g): Lasso

R – Lasso with Cross-Validation
R
library(glmnet)

# Matrix prepare
df_clean <- na.omit(df)
X <- model.matrix(RH ~ ., data = df_clean)[, -1]
y <- df_clean$RH

# Lasso: alpha = 1
set.seed(42)
lasso_cv <- cv.glmnet(X, y, alpha = 1)

# Optimal lambda
best_lambda <- lasso_cv$lambda.min
cat("Optimal lambda:", best_lambda)

# Nonzero coefficients
lasso_coef <- coef(lasso_cv, s = "lambda.min")
print(lasso_coef)

# Ridge: alpha = 0
ridge_cv <- cv.glmnet(X, y, alpha = 0)
ridge_preds <- predict(ridge_cv, s = "lambda.min", newx = X)
ridge_rmse <- sqrt(mean((y - ridge_preds)^2))
cat("Ridge RMSE:", ridge_rmse)
R – Compare OLS vs Ridge vs Lasso
R
# Train/Test split
set.seed(42)
train_idx <- sample(1:nrow(X), 0.7 * nrow(X))
X_train <- X[train_idx, ]; y_train <- y[train_idx]
X_test  <- X[-train_idx, ]; y_test  <- y[-train_idx]

# Ridge MSE
ridge_cv <- cv.glmnet(X_train, y_train, alpha = 0)
ridge_p  <- predict(ridge_cv, s = "lambda.min", newx = X_test)
cat("Ridge Test MSE:", mean((y_test - ridge_p)^2))

# Lasso MSE
lasso_cv <- cv.glmnet(X_train, y_train, alpha = 1)
lasso_p  <- predict(lasso_cv, s = "lambda.min", newx = X_test)
cat("Lasso Test MSE:", mean((y_test - lasso_p)^2))
