Logistic Regression

Binary Classification — probability predict කිරීම. Sigmoid function, Odds, Log-Odds, MLE.

❓ Linear Regression Classification ට ඇයි නොහොඳ?

  • Predictions [0,1] range outside → 1.3, -0.2 possible → Probability invalid
  • Heteroscedasticity (non-constant error variance)
  • Poor interpretability of predicted values
Goal: P(Y = 1 | X) ∈ [0, 1] predict කිරීම. Logistic function guarantees this.

📊 Logistic Function (Sigmoid)

P(Y=1 | X) = 1 / (1 + e^(−(β₀ + β₁X)))

Real values → Maps to [0, 1]. S-shaped curve.

🎲 Odds සහ Log-Odds (Logit)

Odds

Odds = P(Y=1) / (1 − P(Y=1))
  • Odds = 3 → "Success 3x likely"
  • Range: [0, ∞)

Log-Odds (Logit)

log(p/(1−p)) = β₀ + β₁X₁ + ··· + βₚXₚ
  • Linear in X — estimation easy
  • Range: (−∞, +∞)

🔢 Coefficients Interpret

  • βⱼ: Xⱼ 1 unit increase ෙදී log-odds change (other constant)
  • exp(βⱼ) = Odds Ratio: Xⱼ 1 unit increase ෙදී odds multiplicative change
  • exp(βⱼ) > 1 → Odds increase, P(Y=1) ↑
  • exp(βⱼ) < 1 → Odds decrease, P(Y=1) ↓
Interpretation Example
β₁ = 0.5 (Age predictor) → exp(0.5) = 1.65. "1 year age increase ෙදී disease odds 65% increase."

📐 Maximum Likelihood Estimation (MLE)

Logistic Regression parameters OLS ගෙන estimate නොකරයි. MLE use කරයි.

L(β) = Π[π(xᵢ)]^yᵢ × [1 − π(xᵢ)]^(1−yᵢ)

β values ගෙන observed data probability maximize. Iterative numerical methods (Newton-Raphson).

📊 Multiple Logistic Regression

log(p(X)/(1−p(X))) = β₀ + β₁X₁ + β₂X₂ + ··· + βₚXₚ
p(X) = e^(β₀+β₁X₁+···+βₚXₚ) / (1 + e^(β₀+β₁X₁+···+βₚXₚ))

💻 R Code – Logistic Regression

R – Binary Classification
R
# Logistic Regression - disease predict (0/1)
fit_logit <- glm(disease ~ age + cholesterol,
                  data = df,
                  family = binomial("logit"))
summary(fit_logit)

# Odds Ratios
exp(coef(fit_logit))

# Predictions (probability)
probs <- predict(fit_logit, newdata = test, type = "response")

# Classification (threshold 0.5)
preds <- ifelse(probs > 0.5, 1, 0)

# Confusion matrix
table(Predicted = preds, Actual = test$disease)

# Accuracy
accuracy <- mean(preds == test$disease)
cat("Accuracy:", accuracy)
