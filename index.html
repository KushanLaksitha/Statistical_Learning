💡 Exam Tips

🎯 R Code ලිවීමේ Tips

  • lm() — Linear Regression
  • glmnet() — Ridge/Lasso
  • cv.glmnet() — Cross-Validation
  • knn() — KNN
  • Comments: # sign use කරන්න

📊 Key Formulas

  • MSE = mean((y - ŷ)²)
  • Ridge: β² penalty
  • Lasso: |β| penalty
  • Logistic: P = 1/(1+e^(-z))
  • Bias² + Variance + ε²
