Nonlinear Models in Regression

Polynomial, Step Functions, Splines, LOESS, GAMs – curves, thresholds, complex shapes.

❓ Nonlinear Models ඇයි?

ප්‍රශ්නය
Linear Regression straight line assume කරයි. Data හි curves, thresholds, complex shapes ඇතිනම් Linear model poor fit දෙයි. Nonlinear models realistic patterns capture කරයි.
  • Growth of children (height vs. age) — curve
  • Economic data (income vs. education) — threshold effects
  • Medical data (drug dosage vs. effect) — nonlinear response
Q2(b) Exam: Linear vs Nonlinear in Parameters
"Linear in parameters" = β coefficients linear ව appear කරයි (X^2 ඇතත් OK).

i. Yᵢ = β₀ + β₁eˣⁱ + εᵢ → Nonlinear — β₁ exponent X සමඟ nonlinear
ii. Yᵢ = β₀ + β₁Xᵢ^β₂ + εᵢ → Nonlinear — β₂ power parameter (parameter in exponent)
iii. Yᵢ = β₀Xᵢ/(β₁ + Xᵢ) + εᵢ → Nonlinear — β₁ denominator

🛠️ Approaches (Q2(d) – Two Approaches)

1. Transform Predictors

  • Polynomial regression (X, X², X³)
  • Log/sqrt transformations
  • Linear framework maintain

2. Break Predictor Space

  • Step functions (intervals)
  • Splines (piecewise polynomials)
  • Local regression (LOESS)

📐 Polynomial Regression

y = β₀ + β₁x + β₂x² + β₃x³ + ··· + βdxd + ε
  • Linear in parameters β — OLS estimate කළ හැකිය
  • Low degree (quadratic): Simple curves
  • High degree (10th): Overfitting risk!
  • Degree select: Cross-validation, AIC/BIC
R – Polynomial Regression
R
# Degree 3 Polynomial
fit_poly <- lm(y ~ poly(x, 3, raw = TRUE), data = df)
summary(fit_poly)

# Best degree via CV (from rmd.txt)
cv_error <- function(degree) {
  mse <- numeric(K)
  for (k in 1:K) {
    train <- df[folds != k, ]
    test  <- df[folds == k, ]
    fit <- lm(y ~ poly(x, degree, raw = TRUE), data = train)
    pred <- predict(fit, newdata = test)
    mse[k] <- mean((test$y - pred)^2)
  }
  mean(mse)
}
degrees    <- 1:25
cv_results <- sapply(degrees, cv_error)
best_deg   <- degrees[which.min(cv_results)]

🪜 Step Functions

y = β₀ + β₁I(c₁ ≤ x < c₂) + β₂I(c₂ ≤ x < c₃) + ···
  • X range intervals (bins) ලෙස divide කිරීම
  • Each interval constant value
  • Advantage: Simple, interpretable. Natural categories ඇතිනම් useful (age groups)
  • Limitation: Discontinuous jumps at cutpoints. Gradual trends miss

🌊 Regression Splines

Predictor range knots ලෙස divide → Each segment low-degree polynomial → Smoothly join.

Cubic Regression Splines — piecewise cubic, smooth at knots (up to 2nd derivative)

Regression Splines

  • Fixed knots
  • Piecewise polynomials
  • Smooth at knots
  • User selects knots

Smoothing Splines

  • Knot at every data point
  • Penalty λ on curvature
  • λ → smooth/rough tradeoff
  • Data-driven
Smoothing Spline: min Σ(yᵢ − f(xᵢ))² + λ∫[f''(x)]²dx

📍 Local Regression (LOESS/LOWESS)

  • One global model fit නොකොට, each point x₀ ලඟ local weighted regression
  • Nearby points more weight (distance ↑ → weight ↓)
  • Span parameter: Proportion of data for each local fit
  • Small span → more flexible (overfit risk). Large span → smoother (underfit risk)

🔗 Generalized Additive Models (GAMs)

y = β₀ + f₁(x₁) + f₂(x₂) + ··· + fₚ(xₚ) + ε

Multiple predictors, each with smooth nonlinear function fⱼ(xⱼ). Additive structure → interpretability maintain.

Method Characteristics
Polynomial Global curve, risk of oscillation
Step Functions Piecewise constant, simple
Basis Functions General framework for expansion
Regression Splines Piecewise polynomials with knots
Smoothing Splines Data-driven smoothness control
Local Regression Flexible, local fits
GAMs Additive nonlinear models
